# Udemy Search Query Patterns

Pre-built search templates organised by domain. Replace `<TOPIC>` with specific keywords
and `<current-year>` with the actual current year at search time.

## Networking & Infrastructure

```
udemy "<TOPIC>" network engineer automation cisco <current-year>
site:classcentral.com udemy "<TOPIC>" network automation
"<TOPIC>" udemy pyats netmiko nornir review
udemy "<TOPIC>" CCNA CCNP automation python
```

**Common topics:** PyATS, Netmiko, NAPALM, Nornir, Ansible networking, Cisco DNA Center,
Meraki, ACI, SD-WAN, RESTCONF, NETCONF, network programmability

## Cross-Referencing Patterns

To validate course quality, search for independent reviews:

```
"<exact course title>" review reddit
"<instructor name>" udemy instructor review
"<course title>" "worth it" OR "recommended" OR "best"
```

## Instructor Lookup Patterns

When an instructor name is found, search for their full profile:

```
"<instructor name>" udemy instructor courses
"<instructor name>" linkedin network engineer OR developer
site:youtube.com "<instructor name>" <topic>
```

## Date-Aware Searches

Always include the current year to get fresh results:

```
best udemy courses "<TOPIC>" <current-year>
udemy "<TOPIC>" updated <current-year> new content
```

Avoid results older than 18 months unless the course has been recently updated.
